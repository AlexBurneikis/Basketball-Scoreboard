# Controls
spacebar - (pause, unpause, start)  
enter - (reset shotclock)  
p - (+period)  
P - (-period)  

click - (select the team to change) (left or right)  
1,2 and 3 - (+points to selected team)  
!,@ and # - (subtract 1,2 or 3 from selected team's points)  
f - (+foul to selected team)  
F - (-foul to selected team)
